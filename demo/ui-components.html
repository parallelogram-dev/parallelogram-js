Image Lightbox Gallery

Click images to open them in a fullscreen lightbox with navigation. This gallery uses data-lightbox-preload="all" to eager-load all gallery images when opened:

Lightbox Features

  • Gallery Navigation: Arrow keys and click navigation between images
  • Accessibility: Keyboard support with Escape to close
  • Backdrop Control: Click outside to close (configurable)
  • Image Counter: Shows current position in gallery
  • Lazy Loading Integration: Automatically eager-loads lazy images when lightbox opens
  • Preload Strategies: Configure preloading with data-lightbox-preload="adjacent|all|none"
    • adjacent (default): Preload current + previous + next images
    • all: Preload entire gallery for instant navigation
    • none: No preloading (lazy load only)

          Extra Large Modal - Documentation

          PModal Component Documentation

          The PModal component provides a flexible, accessible modal dialog system with the following features:

          Size Options

          • xs (300px): Perfect for simple confirmations and alerts
          • sm (400px): Good for small forms and quick actions
          • md (640px): Default size for standard content
          • lg (800px): Great for forms and detailed content
          • xl (1000px): Ideal for documentation and rich content
          • fullscreen: Full viewport with 3rem inset margins

          CSS Custom Properties

          All styles use em-based CSS custom properties for easy theming and accessibility:

          • Spacing scale: --modal-space-xs through --modal-space-xl
          • Border radius: --modal-radius, --modal-radius-lg, --modal-radius-xl
          • Colors: --modal-panel-bg, --modal-text, --modal-accent
          • Timing: --modal-transition, --modal-animation-duration

          Data Attributes

          Clean separation between styling (BEM classes) and JavaScript hooks (data attributes):

          • data-modal-backdrop, data-modal-panel, data-modal-header
          • data-modal-content, data-modal-footer, data-modal-close-btn
