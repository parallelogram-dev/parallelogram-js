UI Components

Accessible modals, toast notifications, enhanced form controls, data tables, lightbox galleries, and copy utilities.

Enhanced Modal System

Accessible modals with focus management, slot-based content, and multiple sizes:

Modal Features

  • Accessibility: Full ARIA support, focus management, keyboard navigation
  • Flexible Content: Slot-based architecture for title, content, and actions
  • Multiple Sizes: Small, medium, large, and fullscreen options
  • Backdrop Control: Configurable backdrop clicking and escape key handling

Toast Notification System

Toast notifications that integrate seamlessly with the XToasts web component:

Toast Features

  • XToasts Integration: Works with existing XToasts web component
  • AlertManager Bridge: Seamless integration with AlertManager system
  • Event Bus: Emits events for monitoring and logging
  • Graceful Fallback: Falls back to browser alerts if toast system unavailable

Enhanced Select Component

Custom select component with search, accessibility, and form integration:

Select Component Features

  • Native Option Elements: Uses standard HTML option and optgroup elements
  • Form Integration: Full form association with validation support
  • Accessibility: Screen reader support, keyboard navigation
  • Visual Feedback: Disabled states, hover effects, focus indicators

Form Enhancement & Validation

Real-time form validation with custom rules and error messages:

Form Enhancement Features

  • Real-time Validation: Validates as you type with debouncing
  • Multiple Rules: Chain validation rules like required|email|min:5
  • Custom Messages: Override default error messages per field
  • Visual Feedback: Error and success states with CSS classes

Component Architecture Overview

The Enhancement Framework follows a component-based architecture with these key principles:

Progressive Enhancement

All components work without JavaScript and are enhanced when the framework loads.

Event-Driven Communication

Components communicate through a central event bus for loose coupling and monitoring.

Lifecycle Management

The PageManager handles component mounting, unmounting, and cleanup automatically.
